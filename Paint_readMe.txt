Paint.py - python application for drawing simple shapes.

Command syntax:
shape_name color startPoint.x startPoint.y <otherPoints.x otherPoints.y> <radius> LEFT ENTER

Command example:
Polygon red 64 70 63.0 29.0 27.9 56.2 66.7 114.9 89.6 75.6 87.3 75.2 64.3 91.0
Line green 321 136 283.0 177.6
Triangle blue 564 147 570.4 100.2 589.3 103.7
Rectangle green 172 358 11.1 12.6
Circle white 286 528 30.6
ELLIPSE black 626 389 49.0 5.4

Clear button - clearing drawn shapes and commands.
